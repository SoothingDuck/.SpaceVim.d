# .SpaceVim.d

My SpaceVim configuration with the following language layers enabled:
- html
- python

# Installation

## Details for HTML Layer (on Arch)

First install `npm` and `nodejs`:

```bash
sudo pacman -S npm
```

Then lsp for HTML:

```bash
sudo npm install --global vscode-html-languageserver-bin
```
